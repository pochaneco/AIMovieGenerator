# AIMovieGenerator

このプロジェクトはVue 3、Vite、Tailwind CSSを利用したフロントエンドWebアプリのスターターです。

## セットアップ

1. 依存パッケージのインストール
   ```sh
   npm install
   ```
2. 開発サーバーの起動
   ```sh
   npm run dev
   ```

## 主な技術
- [Vue 3](https://vuejs.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
